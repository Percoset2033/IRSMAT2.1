<p align="center">
Что было исправлено и добавлено в версия 2.1:

Интеграция геймификации и мотивации: добавлен мультипликатор за серию правильных ответов, который увеличивает награду и стимулирует игроков на поддержание серии. Реализовано отображение текущей серии, рекорда серий и активных бонусов.

Локальные рекорды и лидерборды: реализована панель рекордов (максимальная серия без ошибок и лучший результат в блиц-режиме), данные сохраняются в localStorage и обновляются после каждой сессии.

Звуковые эффекты (SFX): добавлены короткие звуковые эффекты для правильных и неправильных ответов, что повышает вовлеченность.

Эффект конфетти: реализована анимация конфетти при получении последнего наклейки или Платиновой короны с помощью библиотеки canvas-confetti, для большего визуального поощрения.

Режим "Практика" и "Блиц": добавлены новые режимы — бесконечная тренировка без списывания энергии и времяограниченный режим на 60 секунд с отображением таймера и прогресс-бара.

Обратная связь и поощрения: внедрены мотивационные фразы при правильных ответах, а также возможность делиться успехами через API Web Share.

Темная тема: реализовано переключение на темный режим с сохранением в локальное хранилище.

Работа над ошибками: при неправильном ответе появляется модальное окно с разбором типичных ошибок — расписано шаг за шагом, почему ответ неправильный и что можно сделать лучше. Так же убраны баги со слияние уровней

UI улучшения: добавлена кнопка "Поделиться успехами" в разделе наклеек с использованием стандартизированного API Web Share.
